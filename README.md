# I Learn

### Commit Conventions

| Message | 설명 |
| ------- | --- |
| feat | 새로운 기능 추가  |
| hotfix | 치명적인 버그 수정  |
| fix | 버그 수정 |
| design | CSS 등의 UI 변경 |
| refactor | 코드 리팩토링 |
| docs | 문서 수정 |
| comment | 주석 추가 및 변경 |
| test | 테스트 코드 작업 |
| rename | 파일 혹은 폴더명을 수정하거나 옮기는 경우 |
| remove | 파일 삭제 |

ex : 기능 추가 시 : [feat] member CRUD 구현
